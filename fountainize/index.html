Syntax

Action

The default - if it doesn't follow another rule, it's an action!

This is an action.

Character

Write CHARACTERS in all caps.

SAMATHA, CHARLES

Dialogue

Dialogue always follows a character or paranthetical.

SAMANTHA
I love speaking dialogue!

Scene

Start the line with "int." or "ext."

EXT. Outside the Fairmere Bank

() Paranthetical

Start the line with a bracket

NATHAN
(whispers)

Transition

End the line with TO:

STAR WIPE TO:
